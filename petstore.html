Petstore Database

Description: This project takes a MySQLWorkbench relational database and ties it to a web application to auto update and delete data entries through the application itself. This incorporates client and server side validation that fits the parameters of the database tables.

Role: Solo School Project

Challenges: Getting the database to connect with the application so you can manage it through the web app was a challenge.

Skills: Learning to use JavaScript for client side validation was very useful. Tying the database itself with the application was something we learned as well.
