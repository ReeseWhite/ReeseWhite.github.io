Concert Ticket App

Description: This app gives the user a selection of bands to pick from and then input how many tickets they would like to purchase. After both the band and ticket amount is selected the calculate button will give the user the total price.

Role: School solo project

Challenges: The main challenge with this project for me was the layout design for the app to meet all the requirements while not feeling cluttered. We were given the image and the required information so making sure everything fit and there were no overlapping sections was important. I tried to keep the actual calculation the most important and central part and used the photo to cover extra space without distorting it.

Skills: UI design, Java
